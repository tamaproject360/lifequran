# Assets Directory

Direktori ini berisi semua asset untuk aplikasi LifeQuran:

## Required Assets

### Icons & Images
- `icon.png` (1024x1024) - App icon
- `adaptive-icon.png` (1024x1024) - Android adaptive icon
- `splash.png` (1284x2778) - Splash screen
- `favicon.png` (48x48) - Web favicon
- `notification-icon.png` (96x96) - Notification icon

### Fonts
Letakkan font files di `assets/fonts/`:
- Satoshi-Bold.ttf
- Satoshi-Regular.ttf
- Satoshi-Medium.ttf
- InstrumentSerif-Regular.ttf
- InstrumentSerif-Italic.ttf
- Uthmani.ttf (untuk teks Arab Al-Qur'an)

### Images
- Ilustrasi untuk onboarding
- Badge/achievement icons
- Decorative SVG elements

## Download Fonts

- **Satoshi**: https://www.fontshare.com/fonts/satoshi
- **Instrument Serif**: https://fonts.google.com/specimen/Instrument+Serif
- **Uthmani**: Dari Quran.com atau Tanzil.net

## Notes

Semua asset harus mengikuti design system colors:
- Primary: #22C55E
- Background (dark): #022C22
- Accents: #ADFFD8, #D4AF37
